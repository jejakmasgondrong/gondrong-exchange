# Gondrong Exchange

A Solana-based decentralized exchange (DEX) landing page built from scratch using Next.js and Solana Wallet Adapter.

## Project Overview

This is a personal portfolio project demonstrating modern Web3 development workflows. The application was built manually via "AI-assisted coding" (vibe coding) directly in VS Code, showcasing practical skills in blockchain integration, state management, and modern frontend development.

## Tech Stack

- **Frontend**: Next.js (App Router)
- **Blockchain**: Solana (Devnet only for safety)
- **Wallet Integration**: `@solana/wallet-adapter` (Phantom)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Workflow**: AI-assisted development in VS Code

## Getting Started

### Prerequisites

- Node.js LTS
- Phantom Wallet browser extension (configured to Devnet)

### Installation

```bash
npm install
npm run dev
